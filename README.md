# jym55.github.io
## Publications & Guides Pratiques

Bienvenue sur **jym55.github.io / Typst Notes**, votre ressource en français dédiée au système de composition Typst.

Ce dépôt centralise des notes, des tutoriels, des astuces et des exemples de code pour vous aider à maîtriser la création de documents avec typst.

---

Ce projet est ouvert aux contributions de la communauté francophone ! Si vous souhaitez améliorer une note, signaler une erreur ou proposer une nouvelle publication :

- Ouvrez une **Discussions** et déposez un commentaire dans la catogorie « comments » ;
- Ouvrez une **Discussions** et posez une question dans la catégorie « q-a ».
