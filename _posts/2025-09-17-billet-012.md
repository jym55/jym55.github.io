# Déclarer une page en mode colonnes
Pour avoir une page divisée en deux colonnes, il faut placer le code suivant :

```
#set page(columns: 2)
```

# Définir en plus la hauteur de la page :
Le code est :

```
#set page(columns: 2, height: 350pt)
```

Problème la page est entièrement divisée en deux.

# Insérer un titre par exemple
Le code à ajouter, pour un titre centré en haut (bas) de la page :

```
#place(
  top (bottom) + center,
  scope: "parent",
  float: true,
  text(1.4em, weight: "bold")[
    My document
  ],
)
```
