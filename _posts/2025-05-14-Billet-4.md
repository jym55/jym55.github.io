#### Insertion d'une image

- Pour insérer une image(une photo) qui se trouve au même niveau que le document Typst, il faut utiliser le  code suivant :

    ```
    #image("image1.png")
    ```

- Si le fichier image est dans un sous-dossier contenu dans le dossier du document Typst, il faut modifier le code comme suit :

    ```
    #image("img/image1.png")
    ```

#### Modification de la taille de l'image

Il faut ajouter au code précédent, l'argument « width: xx% ».

Exemple :

```
#image("image1.png", width: 30%)
```

#### Centrer une image

```
#align(center + bottom)[
#image("image2.jpg", width: 70%)]
```
